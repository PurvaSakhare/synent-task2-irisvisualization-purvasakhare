# Iris Data Visualization

## Problem Statement
Analyze and visualize the Iris dataset to identify patterns and relationships among different flower species.

## Dataset
Iris Dataset

## Objectives
- Create meaningful visualizations
- Compare different features
- Identify patterns in the data
- Understand species distribution

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations Created
1. Histogram
2. Scatter Plot
3. Bar Chart

## Steps Performed
1. Loaded the Iris dataset
2. Explored dataset structure
3. Created visualizations
4. Compared different flower features
5. Analyzed species distribution

## Results
- Clear visual representation of feature distributions
- Identification of relationships between sepal and petal measurements
- Better understanding of species characteristics

## Conclusion
The Iris dataset was successfully visualized using multiple chart types, helping identify patterns and differences among flower species.
